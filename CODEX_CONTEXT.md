# CODEX_CONTEXT.md

## Project Name

Website Trust & Security Mini-Audit Service for Italian Small Businesses

## Owner

Juri Buora

## Goal

Build a practical monetization project that helps generate spare income while studying cybersecurity full time.

The business idea is to sell a low-cost website check-up service to small businesses in Italy. The service checks visible and non-invasive issues on already-public websites: basic security configuration, HTTPS, performance, mobile usability, SEO basics, privacy/cookie basics, broken links, and trust/conversion issues.

This is not a penetration test. It must not include intrusive scanning, exploitation, brute forcing, login testing, vulnerability exploitation, directory fuzzing, or aggressive automated testing.

Alternative:

## Core Offer

For €49, the customer receives a simple PDF report with:

- clear findings
- screenshots
- priority levels
- business impact
- practical recommendations
- who should fix each issue

First offer wording:

> Sto facendo i primi audit a prezzo ridotto per creare casi studio: €49.

## Audit Sections

1. First impression and trust
2. Mobile usability
3. Website speed
4. HTTPS / SSL / TLS
5. Header di hardening browser
6. SEO basics
7. Broken links
8. Privacy / cookie basics
9. Contact and conversion path
10. Priority remediation plan

## Tools To Use

Free and non-invasive:

- Google PageSpeed Insights
- Mozilla HTTP Observatory
- Qualys SSL Labs
- Wappalyzer browser extension
- Browser responsive/mobile mode
- Manual SEO checks
- Manual cookie/privacy review
- Manual broken-link review
- macOS screenshots
- Google Docs / Canva / Markdown-to-PDF for reports

Avoid:

- Nessus
- Burp Suite intrusive testing
- sqlmap
- brute forcing
- exploit testing
- aggressive crawling
- login testing
- vulnerability exploitation

## Required Disclaimer

> Nota importante: questo check-up è un'analisi esterna e non invasiva del sito web. Non include penetration test, scansioni aggressive, accesso ad aree riservate, test su login, exploit o verifiche legali complete. Le osservazioni su privacy/cookie non costituiscono consulenza legale, ma indicano elementi visibili che possono essere verificati con il proprio consulente o fornitore web.
