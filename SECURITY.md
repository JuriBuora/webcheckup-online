# Security Policy

## Reporting a vulnerability

Please report a suspected vulnerability privately to [webcheckup.online@gmail.com](mailto:webcheckup.online@gmail.com). Include the affected URL, a concise reproduction path, and enough detail to understand the impact.

Do not use intrusive testing, access private areas, send spam, or disrupt the service. This website is a static, public landing page and does not invite penetration testing.

## Scope

This policy covers the public `webcheckup.online` landing page and its request form. It does not cover websites submitted by prospective clients for a non-invasive check-up.
