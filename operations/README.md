# Operations

This folder is for local-agent business operations.

## Folders

- `prospects/`: research notes for potential clients.
- `outbox/`: email drafts that require human approval before sending.
- `logs/`: local-agent workflow logs.

## Recommended docs

- `one-client-runbook.md`: the simplest lead-to-PDF path for one operator.
- `service-audit-2026-07-03.md`: current business verdict and missing pieces.
- `payment-and-delivery-flow.md`: standard payment and delivery behavior.
- `too-few-findings-policy.md`: what to do when a paid report would be too weak.
- `delivery-log.csv`: minimal delivery log to update after every paid send.

The local agent may write here. It should not send email directly.
